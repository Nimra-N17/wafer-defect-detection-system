# Wafer Defect Detection System

Automated semiconductor wafer inspection system built using Python, OpenCV, and SQLite.

## 🚀 Features
- Automated defect detection using OpenCV
- Image processing pipeline
- SQLite database storage
- Command-line execution
- Modular Python structure

## 🛠 Tech Stack
- Python
- OpenCV
- NumPy
- SQLite
- Git

## ▶ How to Run

```bash
pip install -r requirements.txt
python main.py
```

## 📂 Project Structure

wafer_project/
│── main.py
│── defect_detection.py
│── database.py
│── requirements.txt

## 📊 Output
- Defect count displayed in terminal
- Scan results stored in wafer_scans.db

Developed for automated semiconductor wafer quality inspection.
